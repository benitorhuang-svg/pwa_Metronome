# Metronome Pro - Professional Grade PWA

Master your tempo with a high-fidelity, industrial-grade metronome and tuner. Built with precision and elegance in mind.

🚀 **Live PWA URL:** [https://benitorhuang-svg.github.io/pwa_Metronome/](https://benitorhuang-svg.github.io/pwa_Metronome/)

## ✨ Features

- **Industrial Grade Metronome**: High-precision audio timing with a custom-built worker-based core.
- **Analog-Style Tuner**: Professional semi-circular needle gauge with real-time smooth pitch detection.
- **Glassmorphic UI**: Stunning modern interface with frosted glass effects and dynamic themes (Spring, Winter, Autumn, Cherry Blossom).
- **PWA Ready**: Installable on iOS, Android, and Desktop with offline support.
- **Haptic Feedback**: Physical vibrations synced with the beat for a professional practice experience.
- **Wake Lock Protection**: Prevents the screen from dimming during long practice sessions.
- **CI/CD Integrated**: Automated builds and deployments via GitHub Actions.

## 🛠️ Tech Stack

- **Core**: TypeScript, Vanilla JS (No heavy framework overhead)
- **Styling**: Vanilla CSS (Modern CSS variables, Flexbox/Grid)
- **Audio**: Web Audio API (Oscillator nodes, MediaStream, Analyser)
- **Build Tool**: Vite
- **PWA**: `vite-plugin-pwa`
- **Quality Control**: ESLint, Husky, Lint-staged

## 🚀 Getting Started

### Prerequisites

- Node.js >= 20.0.0
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/benitorhuang-svg/pwa_Metronome.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run in development mode:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
